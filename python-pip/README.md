# Python Flask Demo

A simple Python web application using Flask framework.

## Setup

```bash
pip install -r requirements.txt
```

Or with virtual environment:

```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
```

## Run

```bash
python app.py
```

Visit: http://localhost:5000
